## HTML ENTITIES

----
Entities are used to display
1. Reserved Charecters
2. Special Charecters
3. Emojis
4. Symbols

### Syntax
```
 &entity_name;

    or

&#entity_number;

```

### Example

```
&lt -> right arrow ">"
&gt -> left arrow "<"

```
### Link for HTML Entities
```

https://www.w3schools.com/charsets/ref_html_entities_4.asp

```
